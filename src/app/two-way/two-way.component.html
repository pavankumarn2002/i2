Two-Way

For Two Way Binding

Declare variable in js file of assets/js folder

attribute for input cell-tw=[-tw0]

attribute for element to display input text cell-tw=[-tb0]

attribute for input tway="variable"

attribute for element to display input text innerText="variable"

attribute for input tway="variable"

attribute for element to display input text innerText="variable"

For Two Way Filter

Filter For Array

Declare variable & Array in js file of assets/js folder

attributes for input filter="variable" & filtype="arr"

attribute for element to display filter data fildata="variable"

attribute for element to display error message filerr="variable"

Filter For Array Of Objects

Declare variable & Array of Objects in js file of assets/js folder

attributes for input filter="variable" & filtype="variable.key name"

attribute for element to display filter data fildata="variable"

attribute for element to display error message filerr="variable"
